# Contributing

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/craole-cc/devtools.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes and commit: `git commit -m "your message"`
5. Push to your fork and open a Pull Request

## Guidelines

- **Code Style**: Follow language-specific conventions for each tool
- **Testing**: Include tests for new features or bug fixes
- **Documentation**: Update relevant README files and comments
- **Commit Messages**: Use clear, descriptive messages
- **PR Description**: Explain what your changes do and why

## Structure

This monorepo contains tools across multiple languages. Please:

- Keep changes isolated to their respective tool directories
- Update the root README if adding a new tool
- Reference any related issues in your PR

## Reporting Issues

Use GitHub Issues to report bugs or suggest features. Include:

- Clear description of the issue
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Relevant environment details

## Code of Conduct

Be respectful and constructive in all interactions.

Thank you for contributing!
